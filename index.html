Erlava


A modern take on Erlang

Erlava is a general purpose language for building distributed systems. The language draws inspriation from Erlang and leverages java as the underlying language. Erlava is an interpreted language but it has bytecode support. The language is simple to use and comes packed with a full set of libraries that can be used to build your own applications.
