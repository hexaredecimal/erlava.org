Hello world


This tutorial will guide you through creating a simple "Hello, World!" program in Erlava.


Erlava, inspired by Erlang, utilizes a slightly different syntax for module definitions and comments. Here's a breakdown of the key differences:


  • -opt() and -doc(): Erlava uses -opt() and -doc() annotations at the top level after the module declaration to specify module options and documentation, respectively.
  • Entry Point: The main function serves as the program's entry point.
  • Formatting: Erlava follows C-style formatting, using placeholders like %s for strings and %d for integers. 

      
  % This module defines a simple hello world program
  -module(hello_world).

  % Docstring explaining the module's functionality
  -doc("This module prints 'Hello, World!' to the console.").

  % Entry point of the program
  main() ->
    % Print "Hello, World!" to the console
    io:writeln("Hello, World!").
