Literals


Literals are fixed values that directly appear in your code. They represent raw data and cannot be changed during program execution. They serve as building blocks for more complex data structures and expressions.


Numeric

      
        42  %% Integer literal 
        -100 %% ^^
        2.5  %% Double literal

Atoms

      
        hello world %% An atom is any word starting with a lowercase alphabetic symbol
        these are also atoms %% :)

Boolean

      
        true false %% Look closely, they are actually just atoms.

String

      
        "Hello, world" %% Strings are similar to other languages, especially java.

Lists

      
        [1,2,3,4] %% List can be of any type
        [list, of, atoms] %% ^^
        [] %% Empty list

Map

      
        #{"foo" -> bar, "baz" -> foo} %% This creates a reference, more on those later.

Instance

      
        @Foo {Bar = bar} %% This also returns a reference.
